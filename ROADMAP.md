# NexusPay Platform Roadmap

## v1.0.0 (Current) ✅
- Core payment orchestration flow
- 10 Python microservices
- JWT authentication + RBAC
- Docker Compose deployment
- Basic monitoring (Prometheus + Grafana)
- CI/CD with GitHub Actions

## v1.1.0 (Q3 2026)
- [ ] WebSocket support for real-time payment status updates
- [ ] Webhook delivery system with retry logic
- [ ] Merchant portal enhancements
- [ ] CSV/Excel report export
- [ ] Bulk payment processing API

## v1.2.0 (Q4 2026)
- [ ] 3D Secure (3DS2) authentication
- [ ] Payment links / hosted checkout
- [ ] Subscription/recurring payments
- [ ] Multi-currency settlement
- [ ] Advanced fraud ML models

## v2.0.0 (Q1 2027)
- [ ] Multi-tenant architecture
- [ ] Kubernetes deployment manifests (Helm)
- [ ] Service mesh (Istio)
- [ ] Event sourcing for payment ledger
- [ ] GraphQL API layer
- [ ] Mobile SDKs (iOS, Android)
- [ ] Advanced analytics with ClickHouse
